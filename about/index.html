Road-mapping session


We start each project with a road-mapping session.

This allows us to define S.M.A.R.T. (specific, measurable, attainable, relevant, time-bound) goals for the project.

Using an impact map allows to define features which provide most value and would allow to reach the defined goals faster.

Features being implemented can be converted to code which would provide automatic testing making sure the application behaves as supposed.

Behavior Driven Development (BDD) allows to avoid scope creep and focus on the value.

Document submission application

Project for government agency.

It allows the clients of agency to submit documents without having to wait in lines saving their time.

The clients can be sure that submission would be correct (saving weeks of delay).

Agency workers don't need to do time consuming validation anymore (saving hours of time a day).

As soon as the documents arrive they can start filing them.

Inventory system integration

For a multi-million company an integration between their custom inventory system & on-line shop.

After the integration the on-line shop was taking care of inventory, orders, reservations automatically without any human interaction.

This allowed company to save money as it didn't require time consuming management which was previously done by employees.

It also increased sales by allowing customers to see which products were available.

Sports Coaching application (SaaS)

Teams were doing a lot of tasks manually using pen-and-paper adding to overhead and not having a full view of team health.

Coaches could make better strategic decisions by seeing a full overview of a team. Seeing an analytics for each athlete allowed to assign training sessions for getting most from them.

Players were able to continue with their training on tours as their training sessions were available on their mobile devices. Answering questioners on their wellbeing allowed to customize training sessions for them.

The performance of teams improved greatly allowing them to win more games.
